Flask - chapitre 1 : Hello world !

Bonjour mon cher lecteur ! Commençons dès maintenant à construire des applications Flask. Flask est assez minimaliste pour vous offrir choix et flexibilité ; contrairement aux frameworks plus lourds, vous choisissez ce que vous voulez faire et adaptez Flask pour répondre à vos besoins, tout en étant suffisamment complet pour fonctionner immédiatement.

Nous allons développer ensemble trois applications web : la première est simple et vous permettra de vous familiariser avec Flask et de vous habituer aux nouvelles technologies et terminologies tout en construisant une application web non triviale ; la seconde vous initiera à la création d'une application web utilisant une base de données SQL traditionnelle ; et la dernière, plus complète, exploitera une base de données NoSQL et un framework frontend pour créer une application web utile et esthétique.

Dans ce chapitre, nous verrons brièvement ce qu'est Flask et, peut-être plus important encore, ce qu'il n'est pas. Nous configurerons ensuite notre environnement de développement de base, un serveur web, et installerons un gestionnaire de paquets Python ainsi que Flask lui-même. D'ici la fin du chapitre, nous aurons esquissé notre première application et, comme le veut la tradition ancestrale, nous utiliserons nos nouvelles compétences pour afficher le texte "Hello, World!".

En résumé, nous aborderons les points suivants :

  • Présentation de Flask
  • Configuration de notre environnement de développement
  • Écrire « Hello, World! »

Présentation de Flask

Flask est un micro-framework pour le développement web en Python. Un framework, en termes simples, est une bibliothèque ou un ensemble de bibliothèques qui vise à résoudre une partie d'un problème générique plutôt qu'un cas spécifique complet. Lors de la création d'applications web, certains problèmes doivent toujours être résolus, comme le routage des URLs vers les ressources, l'insertion de données dynamiques dans le HTML et l'interaction avec l'utilisateur final. Flask est un micro-framework car il n'implémente que les fonctionnalités de base (y compris le routage) mais laisse les fonctionnalités avancées (comme l'authentification et les ORM de base de données) à des extensions. Le résultat est une configuration initiale plus simple pour les débutants et plus de choix et de flexibilité pour les utilisateurs expérimentés. Cela contraste avec les frameworks "plus complets" comme Django, qui imposent leur propre ORM et leurs technologies d'authentification. Comme nous le verrons, notre application Hello World en Flask peut être écrite en seulement sept lignes de code, l'application entière tenant dans un seul fichier. Ça vous paraît bien ? Allons-y !

Configuration de notre environnement de développement

Un environnement de développement comprend tous les logiciels qu'un développeur utilise lors de la création d'un programme. Pour commencer, nous installerons un gestionnaire de paquets Python (pip) et le package Flask. Dans ce tutoriel, nous détaillerons les étapes pour développer avec Python 2.7 sur une installation propre d'Ubuntu, mais toutes ces instructions pourront facilement être adaptées à Windows ou macOS.

Installation de pip

Pour notre application Hello World, seul le package Flask est nécessaire, mais nous installerons plusieurs packages Python au cours du développement de nos trois applications. Pour les gérer, nous utiliserons le gestionnaire de packages pip.

Si vous avez développé en Python sans gestionnaire de packages jusqu'à présent, vous apprécierez la simplicité avec laquelle pip permet de télécharger, installer, supprimer et mettre à jour des packages. Si vous l'utilisez déjà, passez directement à l'étape suivante pour installer Flask.

Le gestionnaire pip est inclus nativement dans Python 3.4+ et 2.7.9+. Pour les versions antérieures, une installation manuelle est requise.

Sur GNU/Linux, ouvrez un terminal et exécutez :

sudo apt-get update
sudo apt-get install python-pip

Remarque: Sous Windows ou macOS, téléchargez et exécutez le fichier get-pip.py depuis le site officiel : https://pip.pypa.io/en/latest/installing/#install-or-upgrade-pip.

Et voilà ! Vous pouvez désormais installer n'importe quel package Python via pip.

Installation de Flask

Installer Flask via pip est d'une simplicité exemplaire. Exécutez simplement cette commande :

pip install --user flask

Vous pourrez voir quelques avertissements dans le terminal, mais à la fin, le message Successfully installed Flask devrait apparaître. Vous pouvez désormais importer Flask dans vos programmes Python comme n'importe quelle autre bibliothèque.

Remarque : Si vous utilisez habituellement VirtualEnv pour le développement Python, vous pouvez installer Flask dans un environnement VirtualEnv. Nous aborderons ce point plus en détail dans l'annexe Un aperçu du futur.

Création de "Hello, World!"

Nous allons maintenant créer une page web basique et la servir en utilisant le serveur intégré de Flask sur localhost. Cela signifie que nous allons lancer un serveur web local sur notre machine, auquel nous pourrons facilement envoyer des requêtes depuis cette même machine. Cette méthode est très pratique pour le développement, mais ne convient pas aux applications en production. Plus tard, nous verrons comment déployer des applications Flask avec le serveur web Apache.

Écriture du code

Notre application consistera en un seul fichier Python. Créez un répertoire firstapp dans votre dossier personnel, puis un fichier hello.py à l'intérieur. Dans ce fichier, nous allons écrire le code pour servir une page web affichant le message statique "Hello, World!". Voici le code :

from flask import Flask

app = Flask(__name__)

@app.route("/")
def index():
    return "Hello, World!"

if __name__ == '__main__':
    app.run(port=5000, debug=True)

Détaillons ce que fait ce code : 1. Ligne 1 : Importe Flask depuis le package flask (classique) 2. Ligne 3 : Crée une instance de l'objet Flask en utilisant __name__ comme paramètre. Flask utilise ceci pour résoudre les ressources. Pour nos besoins, __name__ suffit car il lie notre module à l'objet Flask. 3. Ligne 5 : Utilise un décorateur Python. Flask utilise les décorateurs pour le routage des URLs. Ce décorateur signifie que la fonction juste en-dessous (index()) sera appelée quand un utilisateur visitera la page racine ("/") de notre application. Pour les non-initiés : Les décorateurs sont des raccourcis Python puissants qui peuvent sembler magiques au premier abord. En essence, ils modifient la fonction qu'ils décorent. 4.Lignes 6-7 : Définissent une fonction simple qui retourne notre message. Cette fonction est appelée par Flask quand un utilisateur visite notre application, et sa valeur de retour sera envoyée comme réponse. 5.Ligne 9 : Une expression Python commune qui évalue à True si notre application est exécutée directement. Cela empêche le script de s'exécuter accidentellement s'il est importé dans d'autres fichiers Python. 6.Ligne 10 : Lance le serveur de développement Flask en local. Configuration : - Port : 5000 (le port 80 sera utilisé en production) - Debug : True (affiche les erreurs détaillées dans le navigateur)

Exécution du code

Pour lancer notre serveur de développement, ouvrez simplement un terminal et exécutez le fichier hello.py. Si vous avez suivi la structure décrite précédemment, utilisez ces commandes :

cd firstapp
python hello.py

Vous devriez obtenir une sortie similaire à ceci :

* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
* Restarting with stat
* Debugger is active!
* Debugger PIN: 123-456-789

Le processus continue de s'exécuter - c'est notre serveur web en attente de requêtes. Testons-le ! 1. Accès via navigateur : - Ouvrez votre navigateur (Firefox, Chrome, etc.) - Rendez-vous à l'adresse : http://localhost:5000 2. Explications techniques : - localhost : alias de l'adresse loopback (127.0.0.1) - :5000 : port d'écoute (par défaut le HTTP utilise le 80) - Nous utilisons le port 5000 en développement pour éviter les conflits - En production, nous basculerons sur le port 80 standard 3. Résultat attendu : Votre navigateur affichera le message "Hello, World!"

Félicitations, vous venez de créer votre première application Flask !

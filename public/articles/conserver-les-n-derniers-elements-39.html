Conserver les N derniers éléments

Vous souhaitez conserver un historique limité des derniers éléments rencontrés lors d'une itération ou d'un autre type de traitement. Conserver un historique limité est un cas d'usage idéal pour un collections.deque. Par exemple, le code suivant effectue une recherche textuelle simple sur une séquence de lignes et renvoie la ligne correspondante ainsi que les N lignes de contexte précédentes lorsqu'un match est trouvé :

from collections import deque


def search(lines, pattern, history=5):
    previous_lines = deque(maxlen=history)
    for line in lines:
        if pattern in line:
            yield line, previous_lines
            previous_lines.append(line)

# Example use on a file
if __name__ == '__main__':
    with open('somefile.txt') as f:
        for line, prevlines in search(f, 'python', 5):
            for pline in prevlines:
                print(pline, end='')
                print(line, end='')
                print('-' * 20)

Lorsqu'on écrit du code pour rechercher des éléments, il est courant d'utiliser une fonction génératrice avec yield, comme dans la solution de cet article. Cela permet de découpler la logique de recherche du code qui utilise les résultats. L'utilisation de deque(maxlen=N) crée une file de taille fixe. Lorsque de nouveaux éléments sont ajoutés alors que la file est pleine, l'élément le plus ancien est automatiquement supprimé. Par exemple : 

from collections import deque


q = deque(maxlen=3)
q.append(1)
q.append(2)
q.append(3)
print(q)
q.append(4)
print(q)
q.append(5)
print(q)

Bien qu'il soit possible d'effectuer ces opérations manuellement sur une liste (ajouts, suppressions, etc.), la solution utilisant une file est bien plus élégante et nettement plus rapide.

Plus généralement, un deque peut être utilisé chaque fois qu'une structure de file simple est nécessaire. Si vous ne spécifiez pas de taille maximale, vous obtenez une file sans limite de taille permettant d'ajouter ou de retirer des éléments à chaque extrémité. Par exemple :

from collections import deque


q = deque()
q.append(1)
q.append(2)
q.append(3)
print(q)
q.appendleft(4)
print(q)
q.pop()
print(q)
q.popleft()
print(q)

L'ajout ou la suppression d'éléments à chaque extrémité d'une file a une complexité O(1). Ceci contraste avec une liste classique où l'insertion ou la suppression en tête de liste a une complexité O(N).

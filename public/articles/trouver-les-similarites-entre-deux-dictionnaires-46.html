Trouver les similarités entre deux dictionnaires

Vous travaillez avec deux dictionnaires Python et souhaitez identifier rapidement ce qu'ils ont en commun ? Qu'il s'agisse de clés partagées, de valeurs identiques ou de paires clé-valeur communes, il existe des méthodes simples et efficaces pour y parvenir.

a = {
    'x': 1,
    'y': 2,
    'z': 3,
}

b = {
    'w': 123,
    'x': 42,
    'y': 2,
}

Exemples d'opérations

Pour trouver les points communs, il suffit d'utiliser les méthodes keys() ou items() en combinaison avec les opérations d'ensembles (set operations).

Trouver les clés communes

L'opérateur d'intersection & permet de trouver les clés présentes dans les deux dictionnaires.

# Renvoie un ensemble de clés communes :
a.keys() & b.keys()
# Résultat : {'x', 'y'}

Trouver les clés uniques

L'opérateur de différence - permet de trouver les clés présentes dans le premier dictionnaire mais absentes du second.

# Renvoie un ensemble de clés dans 'a' qui ne sont pas dans 'b' :
a.keys() - b.keys()
# Résultat : {'z'}

Trouver les paires (clé, valeur) communes

De la même manière, l'opérateur d'intersection peut être appliqué sur la méthode items() pour identifier les paires identiques.

# Renvoie un ensemble de paires (clé, valeur) communes :
a.items() & b.items()
# Résultat : {('y', 2)}

Manipuler les dictionnaires à partir d'opérations d'ensembles

Ces opérations peuvent également être utilisées pour manipuler ou filtrer des dictionnaires. Par exemple, si vous souhaitez créer un nouveau dictionnaire en excluant certaines clés, vous pouvez le faire avec une compréhension de dictionnaire (dictionary comprehension).

# Créer un nouveau dictionnaire en retirant les clés 'z' et 'w' :
c = {key: a[key] for key in a.keys() - {'z', 'w'}}
# Le dictionnaire 'c' est maintenant : {'x': 1, 'y': 2}

Un mot sur les vues (views)

La méthode keys() d'un dictionnaire ne renvoie pas un ensemble, mais un objet appelé vue de clés (keys-view object). Une fonctionnalité moins connue de ces vues est qu'elles prennent en charge les opérations d'ensembles courantes (union, intersection, différence). Vous pouvez donc les utiliser directement sans avoir besoin de les convertir explicitement en un set.

De même, la méthode items() renvoie une vue d'items (items-view object) qui prend également en charge ces opérations d'ensembles.

Cependant, il est important de noter que la méthode values() ne supporte pas ces opérations. En effet, contrairement aux clés, les valeurs ne sont pas garanties d'être uniques, ce qui rendrait ces opérations d'ensembles moins pertinentes. Si vous devez effectuer ces calculs, vous devrez d'abord convertir les valeurs en un ensemble set.

Les dictionnaires sont des structures puissantes en Python, et savoir les comparer efficacement vous fera gagner en clarté et en performance. En combinant simplement keys(), items() et les opérations d’ensembles, vous avez en main une boîte à outils ultra-rapide pour analyser, filtrer ou croiser vos données.
